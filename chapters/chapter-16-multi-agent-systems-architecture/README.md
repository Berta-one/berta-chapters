# Chapter 16: Multi-Agent Systems Architecture

**Track**: Advanced | **Time**: 10 hours | **Prerequisites**: Chapters 1, 9, 11, 20

---

A single LLM call answers a question; a **multi-agent system** decomposes a goal, assigns sub-tasks to specialised agents, lets them communicate, and composes their results into something no single agent could produce. This chapter builds the architecture from first principles: agent roles and memory, message-passing protocols, coordination strategies (contract-net, blackboards, voting), and the failure modes — deadlock, runaway loops, and cost blow-ups — that bite real deployments. You implement a deterministic agent runtime, a task-allocation auction, and a gossip-consensus algorithm, all offline with NumPy so the behaviour is reproducible.

---

## Learning Objectives

By the end of this chapter, you will be able to:

1. **Model agents** with explicit roles, tools, memory, and a perceive-decide-act loop
2. **Pass messages** with a typed performative protocol (inform / request / propose / accept)
3. **Allocate tasks** with the Contract-Net Protocol and reason about greedy vs optimal assignment
4. **Share state safely** through a blackboard and understand read/write contention
5. **Reach consensus** across agents with gossip averaging and majority voting
6. **Orchestrate workflows** with supervisor, pipeline, and debate topologies
7. **Diagnose failure modes** — deadlock, infinite delegation loops, and runaway token cost
8. **Evaluate** a multi-agent system on task success, cost, and latency, not vibes

## What You'll Build

- **Agent runtime** — a deterministic perceive-decide-act loop with pluggable tools and memory
- **Message bus** — typed performatives routed between agents with a transcript log
- **Contract-Net auction** — managers announce tasks, agents bid, work is awarded to the best bid
- **Gossip consensus** — distributed averaging that provably converges on a connected graph
- **Orchestration patterns** — supervisor, sequential pipeline, and debate, with cost/latency accounting
